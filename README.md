# Identificador de palíndromos
### Basicamente meu primeiro problema de leetcode que estou fazendo que é para identificar se o numero x é um palíndromo.

Pra quem não sabe palíndromo pode ser uma palavra ou número que ser tu botar de traz pra frente fica exatamete igual.
# exemplo: 
- Número:
13131313131 , 1221221221221, 133333333333331, 11.

- Palavra:
Osso, Ovo, Arara, Radar e etc.
